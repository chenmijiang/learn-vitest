# Learn Vitest

一个学习 Vitest 测试框架的示例项目。

## 项目简介

本项目帮助前端开发者快速上手 Vitest，通过分阶段学习计划和实践示例，掌握单元测试、Mock、组件测试等核心能力。

## 项目结构

- `TODO.md` - 分阶段学习计划与打卡清单
- `docs/` - 知识文档
- `wiki/` - llm wiki 层
- `__tests__/` - 示例测试文件

## 快速开始

```bash
# 安装依赖
bun install

# 运行测试
bun run test

# 监视模式
bun run test --watch
```

## 学习路线

1. **阶段 0-1**: 快速上手，跑通最小测试
2. **阶段 2-4**: 掌握测试组织、Mock、环境配置
3. **阶段 5-6**: 组件测试与覆盖率
4. **阶段 7-8**: 工程化与调试
