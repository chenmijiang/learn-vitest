# 初学者路径

## 目标

帮助有前端经验但刚接触 Vitest 的读者先跑通最小闭环，再逐步进入日常开发最常见的能力。

## 建议顺序

1. 先看 [Assertions](../topics/assertions.md)，建立最基本的断言表达。
2. 再看 [Hooks](../topics/hooks.md)，理解 setup 和 cleanup。
3. 接着看 [Environment](../topics/environment.md)，先学会选对环境。
4. 然后进入 [Mocking](../topics/mocking.md)，把最常见的实战能力补齐。
5. 最后看 [Snapshots](../topics/snapshots.md)，判断什么时候适合快照。
